# 📚 School Management API

A Node.js + Express + MySQL API to manage schools and sort them based on distance.

---

## 🚀 Features
- Add new schools
- Fetch schools sorted by proximity
- Distance calculation using Haversine formula

---

## 🛠 Tech Stack
- Node.js
- Express.js
- MySQL

---

```bash
git clone https://github.com/VandanaChauhan01/school-management-api.git
